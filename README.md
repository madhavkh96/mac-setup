### Installing Homebrew

Command:
`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`

### Instaling iTerm2
Command:
`brew install --cask iterm2`

### Installing GitHub
Ref: https://git-scm.com/download/mac 

Command:
`brew install git`

### Installing VSCode
Command:
`brew install --cask visual-studio-code`
